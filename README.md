## 👁️ 123VISION v2.0 - AI Cyber Tracker
**AI-Powered Hand Tracking & Voice Command System**

Aplikasi deteksi gestur tangan real-time menggunakan **MediaPipe AI** dan **Web Speech API**. Didesain dengan antarmuka **Glassmorphism Cyberpunk** untuk pengalaman pengguna yang futuristik.

## 🚀 Fitur Utama
- **AI Hand Tracking**: Melacak 21 titik koordinat tangan secara presisi.
- **Finger Counter**: Menghitung jumlah jari yang terangkat secara live.
- **Voice Commands**: Kontrol sistem menggunakan suara (Bahasa Indonesia). 
  - Ucapkan: *"Mulai"* untuk aktifkan kamera.
  - Ucapkan: *"Berhenti"* untuk reset sistem.
- **Neon Dashboard**: Visualisasi status jari (Thumb, Index, Middle, dll) dengan efek glow.

## 🛠️ Tech Stack
- **AI Engine**: MediaPipe Hands.
- **Language**: JavaScript (ES6+).
- **Frontend**: HTML5, CSS3 (Glassmorphism).
- **Voice System**: WebkitSpeechRecognition API.

## 📦 Instalasi di GitHub Pages
1. Buat repository di GitHub.
2. Upload semua file sesuai struktur folder.
3. Pergi ke **Settings > Pages**.
4. Pilih branch `main` dan klik **Save**.
5. Akses link HTTPS yang diberikan oleh GitHub.
