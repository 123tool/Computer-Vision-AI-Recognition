## 👁️ 123VISION - Cyber Tracker
**AI Hand Tracking & Voice Command System**

Aplikasi deteksi gestur tangan real-time menggunakan **MediaPipe AI** dan **Web Speech API**.

## 🚀 Fitur 
- **AI Hand Tracking**: Melacak 21 titik koordinat tangan secara presisi.
- **Finger Counter**: Menghitung jumlah jari yang terangkat secara live.
- **Voice Commands**: Kontrol sistem menggunakan suara (Bahasa Indonesia). 
  - Ucapkan: *"Mulai"* untuk aktifkan kamera.
  - Ucapkan: *"Berhenti"* untuk reset sistem.
- **Neon Dashboard**: Visualisasi status jari (Thumb, Index, Middle, dll) dengan efek glow.

## 🛠️ Tech Stack
- **AI Engine**: MediaPipe Hands.
- **Language**: JavaScript (ES6+).
- **Frontend**: HTML5, CSS3 (Glassmorphism).
- **Voice System**: WebkitSpeechRecognition API.
